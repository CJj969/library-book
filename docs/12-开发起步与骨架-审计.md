## D. 建议优化项

### O1：命名空间与设计文档不一致

docs/07 §3 使用 `SeatBook/` 作为项目根命名空间，实际项目使用 `LibrarySeatReservation.Web`。没有功能影响（代码能跑），但：
- docs/07 中的 Controller 示例代码用的是 `SeatBook.Controllers`，实际是 `LibrarySeatReservation.Web.Controllers`
- 不一致会让开发者在对照设计文档看代码时产生迷惑

建议在进入 Sprint 3 文档最终化时，在 docs/07 中补充说明"实际项目名称为 LibrarySeatReservation.Web，SeatBook 为设计阶段代号"，或者直接在 Sprint 3 重命名项目以对齐。

### O2：.NET 版本差异（6.0 vs 8.0）

docs/07 §2 约定 .NET 8，实际使用 .NET 6.0。docs/12 已说明原因（macOS 环境限制）。建议在 docs/07 或 README 中标注"开发环境为 .NET 6.0，如使用 .NET 8 需调整目标框架"，确保其他开发者不会因版本不匹配而编译失败。
