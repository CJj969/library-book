- **数据库**: SQLite (macOS 不支持 LocalDB)，文件 `SeatBook.db` 自动生成
- **.NET SDK**: 6.0.428（不是 8.0）
- **EF Core 包**: 6.0.0（Sqlite + SqlServer Design + Tools）
- **dotnet-ef 工具**: 6.0.5 安装成功，但因 arm64/x86_64 架构不兼容无法生成迁移；改用 `EnsureCreated()` 替代

## 已知限制

1. `dotnet-ef` 工具因 arm64/x86_64 架构不兼容无法运行，无法生成 EF Core 迁移文件；使用 `EnsureCreated()` 替代。如需迁移需在 Windows 或 arm64 兼容环境中重新安装工具。
2. SQLite 不支持 `HasFilter()` 过滤索引；冲突检测改为唯一索引 + 应用层双重保障。
3. 所有桩页面返回 `NotImplemented` 信息，实际功能从 Sprint 1 开始。
